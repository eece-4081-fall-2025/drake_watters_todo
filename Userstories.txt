My user stories for my TODO app "five" By: Drake Watters
Epic 1: Task Management

As a user, I want to create a new task so that I can remember things I need to do.
As a user, I want to edit an existing task so that I can correct mistakes or update details.
As a user, I want to delete a task so that I can remove tasks I no longer need.
As a user, I want to mark a task as complete so that I can track my progress.

Epic 2: User Accounts & Authentication

As a user, I want to sign up for an account so that my tasks are saved.
As a user, I want to log in so that I can access my personal to-do list.
As a user, I want to log out so that my information stays private.
As a user, I want to reset my password so that I don’t lose access to my account.

Epic 3: Reminders & Notifications

As a user, I want to set due dates for tasks so that I know when they are due.
As a user, I want to get notifications for upcoming deadlines so that I don’t forget.
As a user, I want to snooze a reminder so that I can delay it until later.

Epic 4: Organization & Categories

As a user, I want to create categories or tags so that I can organize tasks by theme.
As a user, I want to filter tasks by category so that I can focus on one type of task.
As a user, I want to search for tasks so that I can find them quickly.
As a user, I want to reorder tasks by priority so that I can work on the most important first.

Epic 5: User Experience / UI Enhancements

As a user, I want to view tasks in a calendar view so that I can see my schedule visually.
As a user, I want to sort tasks by due date so that I can see what’s urgent.
As a user, I want to use a mobile-friendly interface so that I can manage tasks on the go.
